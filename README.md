## サービス概要

AIを活用したタロット占いを通して、
毎日の気持ちや考えを整理し、心をやさしく整えるWebアプリです。
占いというエンタメ性を取り入れながら、
自分と向き合う時間を楽しく、自然に習慣化できる体験を提供します。

## このサービスへの思い・作りたい理由

私は友人とよくタロット占いに行きます。
その時間は、未来を当ててもらうためだけでなく、自分の気持ちや考えを言葉にすることで心が整理され、
占いが終わる頃には不思議と気持ちが晴れていることが多くありました。

また、未来についての占いをしてもらうことで、
「これからどう生きていこうか」というワクワク感が生まれ、前向きな気持ちになれたことも多いです。

一方で、日常ではAIを使って気持ちや目標を整理することも多くありますが、
「診断されている」「分析されている」という感覚になることがあり、
もう少しやさしく寄り添ってもらえる形があればいいのに、と感じていました。

そこで、AIで解答は行うが、AIに診断されている感覚ではなく、
占い師に相談しているような体験を提供したい
という思いから、このサービスを企画しました。

占いの神秘性と、気持ちに寄り添う言葉を組み合わせることで、
悩みや迷いを抱える人が、自分自身と向き合い、前向きな一歩を踏み出すきっかけになるサービスを目指しています。

## ユーザー層について

ターゲットは 10〜30代の女性 です。

10〜20代前半
占いへの関心が高く、日常の小さな悩みや気分を楽しみながら解消したい層

20〜30代
仕事や人間関係、将来への不安など、内省する時間を求めている層

この世代はスマートフォンでの情報取得やアプリ利用に慣れており、
占いや自己理解コンテンツを日常的に取り入れやすい点も理由の一つです。

## サービスの利用イメージ

今日の運勢（未ログインでも利用可能）

1日1回、タロットカードを引く
カード結果に加えて、今日のこころテーマ、ミニセルフケア（簡単な行動提案）が表示される
占いを通して、その日の気持ちに気づき、やさしく整える体験ができる

感情から占う（ログイン後）
今の気分を選ぶだけで占いができる
質問を考えなくても、感情に寄り添った解釈を受け取れる
言葉にできない気持ちを整理するきっかけになる

ユーザーの気持ち「悲しい」を選択 → 占い師キャラクター「たぬまる先生」がその感情を捉え、
優しく寄り添う占いメッセージを返します。

　例：「疲れた」という気持ちの奥には、
　実は大切なサインが隠れています。

　現れたカード「女教皇」は、
　「直感、知性、内なる声」を通して
　そっとあなたを励ましています。

　無理しなくていいんですよ。
　今日は深呼吸から始めましょう。

占い履歴の保存・振り返り
お気に入り（いいね）登録
過去の結果を通して、自分の心の変化に気づくことができる

## ユーザーの獲得について

Instagram・X（旧Twitter）で占い結果をシェアできる導線を想定
「今日のこころテーマ」やメッセージ性の強い結果により、共感を通じた拡散を狙う
未ログインでも体験できる「今日の運勢」により、心理的ハードルを下げる

## サービスの差別化ポイント・推しポイント

本サービス Fuwaly は、直感的な操作と「感情そのものをテーマにする」点で競合サービスと差別化されています。  
例えば、「神秘のタロットワールド」は多数の占いカテゴリと専門鑑定を売りにしている一方で、選択肢が多く入力負担が大きい構造です。 
また「タロットAI - 当たるカード占い！運勢を毎日ガイド」は AI による詳細な読み取りを提供しますが、自由な質問入力や複雑な占術設定が必要で、初心者には敷居が高い場合があります。 
当アプリは選ぶだけで占えるというシンプルなインターフェースにより、気軽に日常的な心のケアとして使えるUXを提供することを目指しています。  

占いは未来を当てることを目的とせず、今の自分の気持ちに気づき、整理することを中心に設計されています。  
占い結果は答えを押し付けるものではなく、行動や気持ちを見つめ直す「ヒント」として提示され、メンタルケアに近い体験を提供します。  

また、ChatGPTなどの自由度の高いAIと異なり、「何を聞けばいいかわからない」と迷うことがないよう、占いジャンルや感情選択などの導線を用意しています。  
迷わず自然に相談できる体験により、ユーザーが日常的に利用できる自己整理ツールとしての価値を高めています。  

さらに、占い履歴やお気に入り機能により、その時々の気持ちを振り返り、占いを一過性の体験で終わらせず自己理解につなげることが可能です。  

- 占いの種類（本日の運勢・ジャンル別・感情占い）と追加カード機能を組み合わせ、よりパーソナルな体験を提供  
- 結果の履歴保存といいね機能により、過去の占いを振り返りながら自分の傾向を確認できる  
- 占い師キャラクターによる親しみやすい演出で、単なるテキスト占いより没入感が高い  
- 競合アプリでは無料機能が限定的な場合が多いが、本サービスは日常の自己整理ツールとしての価値を提供

## 使用する技術スタック

バックエンド：Rails 7（APIモード）
データベース：PostgreSQL
デプロイ先：Render / Vercel
AI連携：OpenAI API（タロット解説生成）
ライブラリ：Tailwind CSS（UI）、Devise（認証）、Chart.js（履歴可視化）

Reactの基本概念（コンポーネント・props・state・イベントハンドリング）、Next.jsによるページ遷移（Link）やAPI連携（fetch）、フォルダ構造についてを学習済みです。

### タロットカードデータの取得・構築
- 本アプリで使用するタロットカードデータは、マルセイユ版タロットを参考に構築しています。  
- 各カードについて「カード名」「正位置・逆位置の意味」「キーワード」を整理し、AIによる解釈生成に使用できる形に整備しています。  
- 追加カードや引き直しなども、同一占い結果に紐づける設計で、一貫性を保ちながら最大3枚まで構築できます。  

### AI利用上の制限・注意点
- 本アプリの占い結果はあくまで参考として提供されるものであり、医療・法律・金融など専門的な判断を伴う内容には使用できません。  
- 占い結果はAIによる自然言語生成を利用しており、完全な正確性は保証されません。  
- ユーザーが不適切な質問やセンシティブな内容を入力した場合、AIの出力結果が予期せぬ内容になる可能性があります。そのため、不適切な入力に対してはエラーメッセージを表示し、占いを実行できない制御を入れる予定です。  
- 本アプリでは、ユーザーが自分の感情や目標を整理する補助として楽しめるよう設計しており、AIの回答はあくまで「相談相手のような体験」を目的としています。
