## サービス概要

AIを使ったタロット占いで、毎日の気持ちや目標を整理できるサービスです。
占いというエンタメ要素を通して、自分の心と向き合う時間を楽しく提供します。
ちょっとした悩みや迷いも、AIがそっとサポートしてくれる体験を届けます。

## このサービスへの思い・作りたい理由

私自身、日々の気持ちや目標を整理するためにAIに相談することが多くあります。
「ただ相談するだけでなく、もっと楽しく心を整えられたら」と考えたときに、占いという形に行き着きました。
占いは元々好きで、タロットの神秘的なワクワク感は、悩みや迷いを抱える人に寄り添う力があると感じています。
AIと占いを組み合わせることで、日常にちょっとしたドキドキと気づきを提供したいと思い、このサービスを作ろうと思いました。

## ユーザー層について

ターゲットは10〜30代の女性です。
10〜20代前半：占いに興味があり、日常の小さな迷いや好奇心を楽しむ傾向が強い
20〜30代：仕事やプライベートで悩みや目標が多く、自分と向き合う時間を求めている
この層はスマホやアプリでの情報体験に慣れており、占いや自己分析コンテンツに自然に触れやすいという特徴があります。

## サービスの利用イメージ

ユーザーがアプリで今日の気分や悩みを入力
AIがタロットカードを引き、意味をわかりやすく解説
解説をもとにユーザーが気づきや目標を整理
→ 占いを楽しみながら、自己理解やメンタル整理ができる体験が得られます。
毎日短時間で使えるため、日々の習慣として自然に取り入れられます。

## ユーザーの獲得について

SNS（Instagram、TikTok）で占いのスクショをシェアできる機能

## サービスの差別化ポイント・推しポイント

AI×占いの融合：ただのタロット占いアプリではなく、AIがユーザーの入力内容に合わせて解釈をカスタマイズ
自己理解・目標整理をサポート：占いをエンタメとして楽しみながら、心理的気づきや行動指針を得られる
短時間で日常に取り入れられる：数分でできる体験設計により、習慣化しやすい
個人に寄り添う体験：一人ひとりの悩みや目標に合わせた柔軟なカード解説

## 機能候補

- MVPリリース時に作りたい機能
ユーザー登録・ログイン
今日のタロット占い（カード選択＋AIによる解説）
占い結果の履歴保存
シンプルな入力フォームで悩みや気分を入力

- 本リリースまでに作りたい機能
カードテーマ（占い項目）のカスタマイズ
SNSシェア機能
AIによるポジティブなアドバイスや行動提案
デイリーチャレンジや習慣化サポート
占い内容の深堀りモード（より理解を深める）

## 使用する技術スタック

- フロントエンド：React（Next.js 14）
- バックエンド：Rails 7（APIモード）
- データベース：PostgreSQL
- デプロイ先：Render / Vercel
- AI連携：OpenAI API（タロット解説生成）
- ライブラリ：Tailwind CSS（UI）、Devise（認証）、Chart.js（履歴可視化）
