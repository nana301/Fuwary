## サービス概要

AIを活用したタロット占いを通して、
毎日の気持ちや考えを整理し、心をやさしく整えるWebアプリです。
占いというエンタメ性を取り入れながら、
自分と向き合う時間を楽しく、自然に習慣化できる体験を提供します。

## このサービスへの思い・作りたい理由

私は友人とよくタロット占いに行きます。
その時間は、未来を当ててもらうためだけでなく、自分の気持ちや考えを言葉にすることで心が整理され、
占いが終わる頃には不思議と気持ちが晴れていることが多くありました。

また、未来についての占いをしてもらうことで、
「これからどう生きていこうか」というワクワク感が生まれ、前向きな気持ちになれたことも多いです。

一方で、日常ではAIを使って気持ちや目標を整理することも多くありますが、
「診断されている」「分析されている」という感覚になることがあり、
もう少しやさしく寄り添ってもらえる形があればいいのに、と感じていました。

そこで、AIで解答は行うが、AIに診断されている感覚ではなく、
占い師に相談しているような体験を提供したい
という思いから、このサービスを企画しました。

占いの神秘性と、気持ちに寄り添う言葉を組み合わせることで、
悩みや迷いを抱える人が、自分自身と向き合い、前向きな一歩を踏み出すきっかけになるサービスを目指しています。

## ユーザー層について

ターゲットは 10〜30代の女性 です。

10〜20代前半
占いへの関心が高く、日常の小さな悩みや気分を楽しみながら解消したい層

20〜30代
仕事や人間関係、将来への不安など、内省する時間を求めている層

この世代はスマートフォンでの情報取得やアプリ利用に慣れており、
占いや自己理解コンテンツを日常的に取り入れやすい点も理由の一つです。

## サービスの利用イメージ

今日の運勢（未ログインでも利用可能）

1日1回、タロットカードを引く
カード結果に加えて、今日のこころテーマ、ミニセルフケア（簡単な行動提案）が表示される
占いを通して、その日の気持ちに気づき、やさしく整える体験ができる

感情から占う（ログイン後）
今の気分を選ぶだけで占いができる
質問を考えなくても、感情に寄り添った解釈を受け取れる
言葉にできない気持ちを整理するきっかけになる

占い履歴の保存・振り返り
お気に入り（いいね）登録
過去の結果を通して、自分の心の変化に気づくことができる

## ユーザーの獲得について

Instagram・X（旧Twitter）で占い結果をシェアできる導線を想定
「今日のこころテーマ」やメッセージ性の強い結果により、共感を通じた拡散を狙う
未ログインでも体験できる「今日の運勢」により、心理的ハードルを下げる

## サービスの差別化ポイント・推しポイント

1. 「AI診断」ではなく「占い師体験」を重視

本サービスは、AIによる分析ツールではなく、
占い師に相談しているような体験を重視しています。
結果の表現や構成も、評価や断定を避け、寄り添う言葉を大切にしています。

2. メンタルケアに近い占い体験

未来を当てることを目的とせず、
今の自分の気持ちに気づき、整理することを中心に設計しています。
占い結果は答えを押し付けるものではなく、
行動や気持ちを見つめ直す「ヒント」として提示されます。

3. ChatGPTとの明確な違い

ChatGPTは自由度が高い一方で、
「何を聞けばいいかわからない」という状態では使いづらいことがあります。
本サービスでは、占いジャンルや感情選択などの導線を用意することで、
迷わず自然に相談できる体験を提供します。

4. 占い結果を「心の記録」として残せる

占い履歴やお気に入り機能により、
その時々の気持ちを振り返ることができます。
占いを一過性の体験で終わらせず、自己理解につなげられる点が特徴です。
